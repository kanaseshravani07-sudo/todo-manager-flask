# 📝 Todo Manager Flask

🚀 A Modern Flask-Based Todo Management Web Application

---

✨ Built using Flask, Bootstrap 5, SQLite, SQLAlchemy, and Jinja2 focusing on CRUD Operations, Database Integration, Dynamic Rendering, and Full Stack Development concepts.

---

## 🌌 About The Project

Todo Manager Flask is a responsive web application developed to strengthen backend and full-stack development concepts using Flask. The project demonstrates practical implementation of CRUD operations, database integration using SQLAlchemy ORM, dynamic rendering with Jinja2 templating, and responsive UI development using Bootstrap.

The application allows users to efficiently create, update, delete, and search todos while maintaining a clean and interactive user experience.

---

## ⚙️ Features

✅ Create Todos  
✅ Update Existing Tasks  
✅ Delete Todos  
✅ Search & Redirect Functionality  
✅ Responsive Bootstrap UI  
✅ SQLite Database Integration  
✅ Dynamic Rendering with Jinja2  
✅ Beginner Friendly Full Stack Project
