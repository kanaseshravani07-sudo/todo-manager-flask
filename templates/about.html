About This Project

A simple and responsive Todo List Web Application built using Flask and Bootstrap.

📌 About The Website

This Todo List Web App helps users manage their daily tasks efficiently. Users can add, update, delete, and organize tasks easily through a clean and user-friendly interface.

The main purpose of building this project was to improve my practical development skills and gain hands-on experience with backend development using Flask.

⚙️ Tech Stack Used

  • Frontend: HTML, CSS, Bootstrap 5
  • Backend: Python Flask
  • Database: SQLite with SQLAlchemy
  • Templating Engine: Jinja2
  • Version Control: Git & GitHub

🚀 Features

  • Add new tasks
  • Update existing tasks
  • Delete completed tasks
  • Responsive UI using Bootstrap
  • Database integration using SQLite
  • Dynamic pages using Jinja templating
